# Flint No Hitbox

**Spawn TNT with flint and steel - Bypasses anti-cheat!**

## Features
- 🔥 Spawn TNT with flint and steel
- 🛡️ Bypasses most anti-cheat systems
- 🎮 Client-side only - no server mods needed

## Installation
1. Install Minecraft Forge 1.8.9
2. Put the JAR in your `mods/` folder
3. Use flint and steel on any block!

## Building
This repo automatically builds with GitHub Actions.
Download the JAR from Actions → Artifacts.

## Requirements
- Minecraft 1.8.9
- Forge 1.8.9
